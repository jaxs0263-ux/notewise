# Notewise

A clean, single-file note-taking and study application built with HTML, CSS, and JavaScript.

## Features

- 📚 Organize notes by subject
- 📝 Rich note editor
- 🔍 Instant search
- 🎴 Generate study cards
- 💾 Persistent local storage
- 📱 Responsive interface

## Demo

Once GitHub Pages is enabled:

https://jaxs0263-ux.github.io/notewise/

## Getting Started

Clone the repository:

```bash
git clone https://github.com/jaxs0263-ux/notewise.git
cd notewise
```

Open `index.html` in any modern browser.

No build tools or dependencies are required.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
